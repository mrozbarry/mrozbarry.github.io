Cardular is a modern way to play your favorite card games. This is a free, no-bound-rules way to play cards with friends over the internet. Games are persistent, and profiles are tied to your browser so you can come back to your game later.

This isn't just about the standard 52 card deck; you can also create your own custom decks and use those! Why not play Cards Against Humanity, Magic: The Gathering, or Pok√©mon.

Cardular will always be free to play, and free to host. Cardular is built with free (or free-tiered) technologies, meaning if you want to host your own server, you can do it without spending a dime.

Cardular uses the latest web technologies, so make sure you use a modern browser that supports HTML5.

Available Sets

Sets are a way of expressing a set of card decks that are related. For instance, to play Go Fish, you'd just need a single standard 52 card deck with Hearts, Diamonds, Clubs, and Spades, each containing Ace through King, and optionally Jokers.

Sets are defined using a set.json file which describes the number of decks in a set, and some loose rules of how players should interact with them
